Channel Messaging in Python
===========================

# Launch server
> node server.js

# Launch clients

## Basic client
> py client.py

## Client with an IP defined
> py client.py 127.0.0.1

### Client with a name
> py client.py 127.0.0.1 Romain

### Client with AES encryption
> py client.py 127.0.0.1 Romain This_key_for_demo_purposes_only!
